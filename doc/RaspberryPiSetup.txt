Raspberry Pi 2 setup:

1.install Raspbian

2.keyboard_layout

	Raspi-config
		4 Internationalisation Options
		13 Change Keyboard Layout
			Generic 105-key (Intl) PC
			Other
			German
			German
			The default for the keyboard layout
			No compose key
			<No>
	
3.Change Timezone
	Raspi-config
		4 Internationalisation Options
		12 Change Timezone
			Europe
			Berlin
	<Finish>
	sudo reboot

5.autologin
	sudo nano /etc/inittab
		suche nach:  1:2345:respawn:/sbin/getty 115200 tty1
		setze ein # for die Zeile: #1:2345:respawn:/sbin/getty 115200 tty1
		darunter schreibe: 1:2345:respawn:/bin/login -f pi tty1 </dev/tty1 >/dev/tty1 2>&1
		save file (strg+x...)
	sudo reboot

6.hold shift while boot
	Edit config
		comment out the Noobs gens #
		For touchpad 5":
			hdmi_force_hotplug=1
			hdmi_group=2
			hdmi_mode=1
			hdmi_mode=87
			hdmi_cvt 800 480 60 6 0 0 0
			max_usb_current=1
		
		For touchpad 7":
			lcd_rotate=2
		OK
	EXIT
	sudo reboot
